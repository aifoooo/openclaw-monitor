## 🟡 非空断言问题

**位置**: `watcher/index.ts` 多处

**现状**:
```typescript
runEntries.get(entry.runId)!.push(entry);
```

**风险**: 如果 entry.runId 不存在，会抛出运行时错误

**建议**: 使用可选链 + 默认值
```typescript
(runEntries.get(entry.runId) || []).push(entry);
```

**评估**: 当前代码在 push 前有 has() 检查，风险较低
