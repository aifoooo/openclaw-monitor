## 一致性验证方法

### UI 与 API 一致性
```javascript
// 获取 UI 显示的数据
const uiCount = document.querySelectorAll('.chat-item')[0].querySelector('.message-count').textContent;

// 获取 API 返回的数据
const apiData = await fetch('/api/chats/xxx/messages?limit=10000');
const apiCount = apiData.messages.length;

// 比较
uiCount === apiCount → ✅
```

### API 与数据库一致性
```bash
# API 返回
apiCount = curl /api/chats/xxx/messages?limit=10000 | jq '.messages | length'

# 数据库
dbCount = sqlite3 monitor.db "SELECT message_count FROM chats WHERE chat_id='xxx'"

# 比较
apiCount === dbCount → ✅
```

### 文件与数据库一致性
```bash
# 文件
fileCount = jq 'select(.type=="message" and .message.role!="toolResult")' session.jsonl | wc -l

# 数据库
dbCount = sqlite3 monitor.db "SELECT message_count FROM chats WHERE chat_id='xxx'"

# 比较
fileCount === dbCount → ✅
```

### 四方一致性（核心）
```
UI === API === 数据库 === 文件
```
